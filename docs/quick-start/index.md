# Quick Start

This section gets you from a downloaded zip to a working session.

## Basic Flow

1. Extract the release zip.
2. Open **Configurator**.
3. Choose or create a config.
4. Set paths for the tools you use.
5. Choose an autostart mode.
6. Validate and save.
7. Launch Supervisor.

The Configurator is the safest place to make changes. It validates paths, shows warnings, and writes the config for you.

## Recommended First Choice

Use **Terminal Mode** and keep **Use Terminal UI as default interface** enabled unless you specifically want the SteamVR dashboard overlay.

Use **SteamVR Overlay** if you prefer controls inside SteamVR and do not need Terminal UI as your main interface.
