# OSC Issues

## OSC Router Does Not Work

1. Confirm OSC Router is enabled.
2. Check the receive port.
3. Confirm each route is enabled.
4. Make sure another app is not using the same port.

## VRChat Does Not Receive Data

Check VRChat OSC settings and verify the sending app is targeting the expected local port.

## OscGoesBrrr / Intiface Does Not Start

1. Confirm paths in **OSCGoesBrrr**.
2. Confirm process names.
3. Use Terminal UI or the classic console action to start the workflow manually.
