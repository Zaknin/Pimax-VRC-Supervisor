# Base Station Issues

## Base Stations Do Not Wake

1. Confirm base-station automation is enabled.
2. Confirm the station is enabled in the Base Stations table.
3. Make sure Bluetooth is available.
4. Try **Refresh State** or a manual power action from Configurator.

## Base Stations Do Not Sleep

Check the selected power-down mode and confirm cleanup is actually running at session end.

## State Looks Wrong

Some stations may not report state reliably. Use manual testing and SteamVR state as a cross-check.
