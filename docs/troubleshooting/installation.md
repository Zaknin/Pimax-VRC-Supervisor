# Installation Issues

## Windows Blocks The App

If Windows warns about an unknown app, confirm you downloaded it from the intended release page and extracted the full zip.

## Configurator Does Not Start

Try these checks:

1. Confirm `PimaxVrcSupervisorConfigurator.exe` exists.
2. Extract the zip again into a writable folder.
3. If using a smaller package, install the required .NET Windows Desktop Runtime.

## Files Are Missing

Do not run directly inside the zip viewer. Extract the full folder first.
