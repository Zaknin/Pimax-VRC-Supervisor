# Troubleshooting

Start with the symptom that matches what you see.

## Quick Checks

1. Open Configurator.
2. Click **Validate**.
3. Confirm paths point to existing executables.
4. Confirm the selected **Autostart mode** is what you expect.
5. Save again if you moved the release folder.
