# OscGoesBrrr / Intiface

Supervisor can help start Intiface and OscGoesBrrr during a session.

## Configure

Open **OSCGoesBrrr** in Configurator.

Set:

- OscGoesBrrr executable
- process names
- startup delay
- whether to use manual launch mode
- optional BLE detection settings

## Recommended Use

If you only use OscGoesBrrr sometimes, keep manual launch mode enabled and start it from Terminal UI or the classic console when needed.

If you want automatic startup based on device detection, configure the BLE scanning options and test carefully.
