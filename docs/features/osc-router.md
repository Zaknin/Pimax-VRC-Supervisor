# OSC Router

OSC Router helps route local OSC messages between VRChat, face-tracking tools, and configured targets.

## Configure

Open **OSC Router** in Configurator.

Set:

- whether OSC Router is enabled
- the local receive port
- destination routes

## When To Use It

Use OSC Router when multiple tools need to send or receive OSC messages without manually changing ports every session.

## Troubleshooting

If OSC messages do not arrive, check ports, route enabled state, firewall prompts, and whether another app is already using the same port.
